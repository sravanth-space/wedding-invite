# wedding-invite (React)

This project is now a React + Vite wedding invite app.

## Run locally

```bash
yarn install
yarn dev
```

## Build

```bash
yarn build
yarn preview
```

## Deploy to GitHub Pages

This repo is set up for GitHub Pages via GitHub Actions.

1. Push your changes to `main`.
2. In GitHub, open `Settings` > `Pages`.
3. Set `Source` to `Deploy from a branch`.
4. Select `gh-pages` and `/ (root)`.
5. After the workflow completes, open the Pages URL from the repository settings.

## Easy configuration

Edit one file:

- `src/config/inviteConfig.js`

You can quickly change:

- Couple names and headings
- Home section date/time/location
- Bride and groom details
- Events list (reception/wedding or more)
- Countdown target date
- Invitation text
- Gallery image lists
- Closing text
- Music and decorative image URLs
